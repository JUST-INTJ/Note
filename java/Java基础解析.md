## 类型擦除

Java 编译器会将泛型代码中的类型完全擦除，使其变成原始类型。

当然，这时的代码类型和我们想要的还有距离，接着 Java 编译器会在这些代码中加入类型转换，将原始类型转换成想要的类型。这些操作都是编译器后台进行，可以保证类型安全。

总之泛型就是一个语法糖，它运行时没有存储任何类型信息。
