# 笔记风格

- 叙述风格：
  - 尽量避免写大段说明
    - 基本上都会把知识点以列表或者表格的形式展示
    - 并且标注每一个知识点的划分，比如`原因`,`作用`,`原理`等等。
  - 列表下面会有引用作为补充说明

- 段落风格：
  - 尽量控制在五级标题以内。
  - 标题内内容的划分主要有两种
    - `---`:以前经常用
    - `> **内容说明**`:现在比较常用，有时间会把`---`都改成这种风格
      > 注意，引用不加粗是补充说明。引用加粗是段内划分

- 待做项说明：
  - 待做项会以`待`字为开头。放在一行。比如`待整理`，`待补充`
  - 通常会罗列一些资料在下面
